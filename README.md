This is a full-stack React/Flask application currently in development, designed to help musicians track creative sessions, organize musical ideas, and build a structured log of their created music.

## Overview

The goal of this project is to create a simple, intuitive platform where musicians can log sessions, capture ideas, and revisit past work - bridging the gap between creative exploration and organized tracking.

## Tech Stack

- Frontend: React (Vite)
- Backend: Flask + SQLAlchemy
- Database: SQLite (with plan to updagrade to PostgreSQL)

## Current Functionality

- Backend data models for sessions and musical entries
- API routes for creating and retrieving session data
- Frontend components for displaying and interacting with session information
- Basic structure for tracking and organizing musical ideas

## In Progress

- Improving UI/UX for session and musical idea logging
- Expanding features for categorizing/filtering and revisiting/editing ideas
- Enhancing frontend-backend data flow and state management

## Purpose

This project was built to strengthen my full-stack development skills and deepen my understanding of how frontend interfaces connect with backend systems and databases.

It also reflects my background in music, combining technical development with a domain I’m personally passionate about.
